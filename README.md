# ProСвітло - База адрес

Статична база даних адрес Хмельницької області для системи моніторингу відключень електроенергії.

## Статистика

- **Населених пунктів**: 1,243
- **Вулиць**: 16,786
- **Будинків**: 343,844
- **Розмір**: 76 MB
- **Останнє оновлення**: 2 грудня 2025

## Структура

```json
{
  "Назва міста": {
    "Назва вулиці": {
      "Номер будинку": {
        "queue": "Черга відключення",
        "source_url": "Джерело даних"
      }
    }
  }
}
```

## Джерела даних

Дані зібрані з офіційних Excel файлів АТ "Хмельницькобленерго":
- https://hoe.com.ua/page/pogodinni-vidkljuchennja

## Використання

Backend сервер завантажує цей файл при старті для швидкого пошуку адрес та черг відключень.
